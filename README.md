## Overview
CampusCloud is an academic group project focused on designing a scalable
and secure university database system. The project emphasizes data modeling,
normalization, and SQL-based data retrieval.

This repository contains the full set of project artifacts, including
documentation, diagrams, and sample SQL queries.
